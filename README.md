# SeedFormer
SeedFormer-频泰科技实习牙齿补全模型仓库

- 原文仓库链接：https://github.com/hrzhou2/seedformer

## Installation

The code has been tested on one configuration:

- python == 3.6.8
- PyTorch == 1.8.1
- CUDA == 10.2
- numpy
- open3d

```
pip install -r requirements.txt
```

Compile the C++ extension modules:

    sh install.sh
